#  Registro de Errores y Deuda Técnica (ErroresREADME)

Este documento rastrea los problemas conocidos actuales en la aplicación web.

---

### 1. Carga inicial lenta en dispositivos móviles (Web)
**Descripción del problema:** 
Al acceder a la aplicación mediante el enlace web desde un navegador móvil, la pantalla se queda en blanco por un tiempo prolongado antes de mostrar la interfaz.



### 2. Desconexión en la carga de imágenes de productos
**Descripción del problema:** 
En el módulo de registro/edición de productos, la funcionalidad para cargar y guardar imágenes no está operativa ni vinculada a los artículos.



### 3. Lista de inventario invisible en pantallas móviles
**Descripción del problema:** 
En el módulo de inventarios, la lista de productos desaparece por completo al visualizar la app en un dispositivo móvil. Los elementos superiores (valor total, unidades, alertas, buscador y categorías) se renderizan sin problemas.
